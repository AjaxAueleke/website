Q.Write about issues and/or observations that helped you install Java 8.

The task only required me to install Java 8 so it was fairly easy. I think the reason I found this easy is that I have worked with several languages like C, C++, Python, Javascript, and MATLAB so I found the installation procedure familiar. The other reason could be that I used to play Minecraft which requires you to install Java so I guess that made it pretty easy. The only issue I faced initially was that I could not understand why I had to use Java SE Development Kit 8, while the 16 is the latest version but I guess you guys required it for some reason so I went with it.

PS : I could not find the exact jdk-8u281-windows-x64.exe so I used jdk-8u291-windows-x64.exe.
